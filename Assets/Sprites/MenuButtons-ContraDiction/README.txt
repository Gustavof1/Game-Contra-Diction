=====================================
Neon Pink Arcade UI Pack – README
=====================================

Author : Uryon Games
Format : PNG (+ optional .pxo sources)
Resolution :
- Small icons : 24x24
- Medium buttons : 64x24
- Large buttons : 100x24
Exported at 6x scale for pixel-perfect rendering.
Color theme : Neon pink / retro arcade pixel-art style


---------------------------------------
CONTENTS (Total : 45 UI Buttons)
---------------------------------------

1) LABELED BUTTONS (9)
- Back
- Exit
- Next
- Start
- Play
- Quit
- Retry
- Settings
- Options

2) BLANK BUTTONS (3)
- Small blank button
- Medium blank button
- Large blank button

3) ICON BUTTONS (3)
- Play icon
- Pause icon
- Settings icon

4) STATES (for every button)
- Normal
- Hover
- Pressed

5) FORMATS
- .png ready for Unity, Godot, GameMaker, etc.
- .pxo source files included (editable in Pixelorama)


----------------
HOW TO USE
----------------
1. Import the PNG files into your engine.
2. Use:
   - "Normal" for default state
   - "Hover" when the button is selected or focused
   - "Pressed" when the player clicks or taps
3. (Optional) Customize colors or labels using the .pxo source files.
